# Shell_Gen

Python 3 based Reverse Shell Generator

Shell Gen generates reverse shells with the following methods:
		
*PHP*  
*Bash*  
*Perl*  
*Ruby*  
*Python*  
*NetCat*  
*Telnet*  
*PowerShell*  

### Run the script
`./Shell_Gen.py <LHOST> <LPORT>`
